#  TP Biblioteca - Programación III 

¡Bienvenidos al repositorio del Trabajo Práctico de Biblioteca! Este proyecto fue desarrollado para la cátedra de **Programación III** de la **Tecnología en Sistemas de Información (UNLaR)**.

##  El Desafío
El objetivo es gestionar el sistema de préstamos de una biblioteca aplicando los conceptos fundamentales de la **Programación Orientada a Objetos (POO)** en Java. 

###  Tecnologías y Conceptos Aplicados:
* **Java 17+**: Lenguaje principal.
* **POO Avanzada**: Uso de Herencia, Encapsulamiento y Polimorfismo.
* **Gestión de Excepciones**: Creación de alarmas personalizadas (Custom Exceptions) para validar reglas de negocio (ej: libros no disponibles o alumnos con deudas).
* **Colecciones (Java Collections)**: Uso estratégico de `ArrayList`, `HashSet` y `HashMap` para manejar el catálogo y los socios de forma eficiente.
* **API de Fechas**: Implementación de `java.time.LocalDate` para el control de vencimientos.
* **Recursividad**: Algoritmos optimizados para cálculos de fidelidad o estadísticas.

##  Estructura del Proyecto
El código está organizado de manera modular:
1. **Entidades**: Clases como `Libro`, `Estudiante` y `Prestamo`.
2. **Lógica de Negocio**: El motor que conecta todo y valida los préstamos.
3. **Excepciones**: Clases que manejan los errores específicos del sistema.
